# AIOS

**Artificial Intelligence Operating System**

AIOS is a modular autonomous engineering operating system designed to help the founder build, monitor, repair, document, and evolve multiple software projects.

## Status

Current phase:

- ✅ Phase -1 — Architecture Decision Records
- ✅ Phase 0 — Foundation
- 🔜 Phase 1 — Core Architecture

## North Star

> Build once. Evolve forever.

## Repository Structure

```text
docs/
  codex/          AIOS Codex volumes
  adr/            Architecture Decision Records
  foundation/     Phase 0 foundation documents
  architecture/   Core architecture documents
  organization/   Agent and department organization
  sdk/            Agent, plugin, department, project SDK
  operations/     DevOps, monitoring, incident, rollback
  api/            API specifications
  database/       Database design
  pixel-office/   Pixel Command Center documentation
  future/         Future vision and innovation backlog
  research/       Research notes
templates/        Document templates
implementation/   Future implementation notes/code

```

## Working Principle

Foundation first. Features later.

Every major decision must be traceable to ADR and Codex.
