# Contributing to AIOS

## Document Rules

Every document must include:

- Document ID
- Title
- Status
- Phase
- Owner
- Dependencies
- Related documents

## Commit Style

Use clear commit messages:

```text
phase(-1): add ADR wave 1
phase(0): add AIOS Constitution
architecture: add kernel overview
portal: update navigation
research: add innovation proposal
```

## Branching

Recommended:

```text
main     stable
develop  active work
```
