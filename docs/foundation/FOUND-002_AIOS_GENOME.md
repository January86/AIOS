# AIOS Genome

AIOS Genome is the inherited behavioral, operational, and architectural DNA used by every agent, department, workflow, and plugin.

## Layers

- Constitution Layer
- Decision Layer
- Communication Layer
- Memory Layer
- Learning Layer
- Tool Layer
- Reporting Layer

## Agent Template

Each agent must define:

- name
- role
- mission
- allowed tools
- memory access
- autonomy level
- KPIs
- escalation path
- reporting format
- failure mode

## Rule

New agents must not be created from random prompts. Every agent must inherit AIOS Genome.
