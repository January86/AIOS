# AIOS PRD v1

## Product Summary

AIOS is a modular autonomous engineering operating system that helps the founder build, monitor, repair, document, and evolve multiple software projects.

## MVP Goals

1. Register projects.
2. Monitor project health.
3. Detect crashes/errors.
4. Create tasks from events.
5. Assign agents.
6. Report status.
7. Create suggested fixes.
8. Enforce autonomy policy.
9. Maintain memory.
10. Visualize agent state.
