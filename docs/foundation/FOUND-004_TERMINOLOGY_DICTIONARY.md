# AIOS Terminology Dictionary

## Terms

**AIOS** — Artificial Intelligence Operating System.

**Founder** — Human owner and strategic authority.

**Mission** — High-level objective executed by AIOS.

**Task** — Smallest actionable unit of work.

**Project** — Software system or operational asset managed by AIOS.

**Module** — Project representation inside AIOS.

**Plugin** — Capability extension installed into AIOS.

**Agent** — AI-operated role with mission, skills, tools, memory, policy, and state.

**Department** — Group of agents with shared responsibility.

**Workflow** — Repeatable process used to execute missions or tasks.

**Event** — Meaningful system occurrence that may trigger workflows.

**Policy** — Rule that allows, denies, escalates, or restricts actions.

**Memory** — Structured retained knowledge.

**Codex** — Single source of truth for AIOS.
